Web Socket chat
===========
- Python - 3.6.3
- Django - 2.0.2
- Redis - 3.0.6
- channels - 2.0.2
- channels-redis - 2.1.0

Install 
-------  
```sh
$ virtualenv -p python3.6 envname
$ source envname/bin/activate
$ git clone
$ cd WS_Chat
$ pip install -r requirements.txt
$ python manage.py runserver
```
